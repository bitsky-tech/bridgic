# 欢迎来到 Bridgic 的世界！

**Bridgic** 是一个创新的编程框架，旨在创建代理系统，从确定性工作流到自主代理。它引入了一种新的范式，简化了代理系统的开发。

## “*Bridgic*” 这个词是什么意思？

名称 **Bridgic** 的灵感来源于“**桥接逻辑与魔法**”的理念——将逻辑的精确性与智能的创造力结合在一起。

- **逻辑** 代表确定性执行流程。
- **魔法** 代表高度自主的 AI 代理。
